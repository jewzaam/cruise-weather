# Cruise Weather — Project Standards

Project standards for `org.jewzaam.cruiseweather`. These will eventually merge upstream into
[jewzaam/standards](https://github.com/jewzaam/standards).

## Guiding Principles

1. **Consistency** — Same patterns throughout. If you do it one way somewhere, do it that way everywhere.
2. **Simplicity** — The right amount of complexity is the minimum needed for the current task.
3. **Automation** — If it can be checked automatically, it should be. Manual checks are unreliable.
4. **Discoverability** — New contributors should be able to find what they need without asking.
